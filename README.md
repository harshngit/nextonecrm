# Next One Realty CRM

A full-featured Real Estate CRM platform built with React + Vite + Tailwind CSS.

## Features

- 🏠 **Dashboard** — Stats, lead funnel, activity feed, team performance
- 👥 **Leads** — Full lead management with filters, status tracking, add/edit
- 📋 **Lead Detail** — Timeline, notes, status updates, quick actions
- 📅 **Site Visits** — Calendar & list view, schedule visits
- 📞 **Follow-Ups** — Today/upcoming/overdue task dashboard
- 🏗️ **Projects** — Project cards with sold unit tracking
- 👤 **Team** — Member management with role-based display
- 🔔 **Notifications** — Grouped, read/unread state
- 🌙 **Dark Mode** — Full dark/light theme support
- ⚡ **Loaders** — Page loader, card skeleton, list skeleton

## Tech Stack

- React 18 + Vite
- React Router v6
- Tailwind CSS (dark mode, custom colors)
- lucide-react icons
- DM Sans + Playfair Display fonts

## Setup & Run

```bash
npm install
npm run dev
```

## Login

Use any email/password combination. Select your role from the dropdown.

## Brand Color

Primary: `#b1916c` (warm gold-brown)
